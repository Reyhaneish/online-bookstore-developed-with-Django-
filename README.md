# online-bookstore-developed-with-Django-
This project is an online bookstore developed with Django. It allows users to search, view, and purchase their desired books.
Advanced Search: Search books by title and description.
Book Details Display: Show book information including title, cover image, price.
Recommended books section: Show books in the same category.
in home page: Show new ,featured and most visited products.we also have a category tab in this page.
Shopping Cart: Add books to the shopping cart.
User Account Management: Create and edit user accounts.
we also have aboutus and contactus page.
Admin Panel: Manage books, categories, etc.
