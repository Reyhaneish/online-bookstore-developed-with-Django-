# online-bookstore-developed-with-Django-<br>
This project is an online bookstore developed with Django. It allows users to search, view, and purchase their desired books.<br>
Advanced Search: Search books by title and description.<br>
Book Details Display: Show book information including title, cover image, price.<br>
Recommended books section: Show books in the same category.<br>
in home page: Show new ,featured and most visited products.we also have a category tab in this page.<br>
Shopping Cart: Add books to the shopping cart.<br>
User Account Management: Create and edit user accounts.<br>
we also have aboutus and contactus page.<br>
Admin Panel: Manage books, categories, etc.<br>
